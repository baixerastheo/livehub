# LiveHub — Backend

NestJS REST API + WebSocket gateway for the LiveHub platform.

See the [root README](../README.md) for full documentation, architecture, and setup instructions.

## Quick start

```bash
pnpm install
cp .env.example .env   # fill in your values
pnpm run start:dev
```

API docs available at `http://localhost:4001/api` (Swagger) once running.
